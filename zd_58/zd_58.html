1)Co to są pseudoklasyk?


Pseudoklasy w HTML są specjalnymi selektorami, które pozwalają nam określić i stylizować elementy na podstawie ich stanu lub pozycji wewnątrz dokumentu. Przykłady pseudoklas to ":hover" (gdy kursor znajduje się nad elementem), ":active" (gdy element jest wciśnięty), ":first-child" (gdy element jest pierwszym dzieckiem swojego rodzica) itp. Pseudoklasy pozwalają na bardziej precyzyjne określanie stylów dla konkretnych przypadków.


2)Uwaga dotycząca dwukropka :


Dwukropek (:) w HTML używany jest do oznaczania pseudoklas, które są specjalnymi selektorami, jak np. ":hover", ":active", ":first-child".


3)Tabela(wykonana w htmlu) opisującą pseudoklasy


Pseudoklasa Opis
:hover Gdy kursor znajduje się nad elementem
:active Gdy element jest wciśnięty
:focus Gdy element ma aktywne skupienie
:visited Gdy odwiedziliśmy już link
:first-child Gdy element jest pierwszym dzieckiem
:last-child Gdy element jest ostatnim dzieckiem
:nth-child(n) Gdy element jest n-tym dzieckiem
:nth-of-type(n) Gdy element jest n-tym elementem danego typu
:not(selector) Gdy element nie pasuje do określonego selektora

4)Zastosowanie pseudoklasy :active


Pseudoklasa `:active` w HTML jest stosowana do stylizacji elementu, gdy jest on aktywny, czyli w momencie, gdy użytkownik wciśnięty trzyma przycisk myszy na elemencie. Dzięki tej pseudoklasie możemy zmieniać wygląd elementu podczas trwania interakcji użytkownika, co może wpływać na jego responsywność i interaktywność. Na przykład, możemy zmienić kolor tła lub obramowania elementu, aby wskazać, że jest on wciśnięty. Pseudoklasa `:active` jest szczególnie przydatna przy tworzeniu przycisków i interaktywnych elementów w interfejsie użytkownika.


5)Przykłady (trzy) kodów definiowania pseudoklas (wraz z tematem)


1. Przykład z pseudoklasą :hover i tematem "zmiana koloru tła po najechaniu":
< style>
.hover-element:hover {
background-color: yellow;
}
< /style>
< div class="hover-element">
Element, na którym zmienia się kolor tła po najechaniu
< /div>
W tym przykładzie używamy pseudoklasy :hover, aby zmienić kolor tła elementu na żółty, gdy kursor myszy znajduje się nad nim.

2. Przykład z pseudoklasą :nth-child i tematem "stylizacja co drugiego elementu": < style>
li:nth-child(even) {
background-color: lightgray;
}
< /style>
< ul>
< li>Element 1< /li>
< li>Element 2< /li>
< li>Element 3< /li>
< li>Element 4< /li>
< li>Element 5< /li>
< /ul>
W tym przykładzie używamy pseudoklasy :nth-child(even), aby stylizować co drugi element listy < li> poprzez zmianę koloru tła na jasnoszary.

3. Przykład z pseudoklasą :visited i tematem "zmiana koloru odwiedzonych linków":
< style>
a:visited {
color: purple;
}
< /style>
< a href="https://www.example.com">Odwiedź stronę< /a>

W tym przykładzie używamy pseudoklasy :visited, aby zmienić kolor odwiedzonego linku na fioletowy. Po kliknięciu w link i powrocie na stronę, link będzie miał zmieniony kolor.
